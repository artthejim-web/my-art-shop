# Jim Mbugua Art - Portfolio Website

Welcome to the Jim Mbugua Art portfolio website! This is a professional art shop showcasing hyperrealistic portraits, animal portraits, sketches, and custom commissions.

## 🎨 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Gallery Section**: Display your artworks with title, medium, and size
- **About Page**: Share your story and upload a profile picture
- **Contact Page**: Direct links to Instagram and WhatsApp
- **WhatsApp Integration**: Direct ordering through WhatsApp from gallery items
- **Smooth Navigation**: Easy navigation between all sections

## 🚀 Setup Instructions

### 1. Add Your Profile Picture
- Replace `profile.jpg` with your own picture
- Place it in the root directory
- Recommended size: 400x400px or larger

### 2. Add Your Artworks
- Add your artwork images to the root directory
- Name them: `artwork1.jpg`, `artwork2.jpg`, etc.
- Recommended size: at least 600x500px
- Update the gallery items in `script.js` with your actual artworks

### 3. Update Gallery Items (Optional)
Edit the `galleryItems` array in `script.js` to match your artworks:

```javascript
const galleryItems = [
    {
        title: "Your Artwork Title",
        medium: "Charcoal",
        size: "A3 (297 x 420 mm)",
        image: "artwork1.jpg"
    },
    // Add more items as needed
];
```
